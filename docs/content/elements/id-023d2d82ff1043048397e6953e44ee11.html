DefinitieEen identiteitsgegeven dat een entiteit uniek identificeert.
Alternatieve termdigitale identiteit
ToelichtingDit is in veel gevallen een pseudoniem. Er kunnen ook andere gegevens die gezamenlijk gebruikt kunnen worden om een entiteit te identificeren. Dit noemen we geen identifiers. Er hoeft geen identifier onderdeel te zijn van de verzameling identiteitsgegevens bij een identiteit.
